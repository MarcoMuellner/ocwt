# ocwt docs

`ocwt` helps you spin up and manage Git worktrees fast, with optional planning in OpenCode before implementation.

## Start here

1. Install: [installation](installation.md)
2. Run your first flow: [quickstart](quickstart.md)
3. Learn command details: [commands/open.md](commands/open.md)

## Core ideas

- `open` creates or reuses a worktree for an intent, branch, or file.
- `close` removes the worktree and deletes the local branch safely.
- planning mode can run first and then continue in the same session.
